# Архив · Дизайн

Черновики-мокапы арт-направлений редизайна (июль 2026), из которых выбрана
дизайн-система **«АЛЬБОМ»** (см. скилл `wc2026-design`). Standalone HTML —
открываются в браузере без сборки. В деплой не попадают (лежат вне `public/`).
